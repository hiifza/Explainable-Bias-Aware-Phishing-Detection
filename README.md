## 📌 Project Overview

Phishing websites remain one of the most prevalent cybersecurity threats, exploiting deceptive URLs, domain characteristics, and webpage behaviors to trick users into revealing sensitive information.

This project aims to develop a **transparent, interpretable, and bias-aware phishing website detection system** that not only identifies malicious websites with machine learning but also explains *why* a website has been classified as phishing.

Unlike traditional black-box models, this project focuses on:

* 🧠 Explainable Artificial Intelligence (XAI)
* 🔍 Cybersecurity Analytics
* ⚖️ Fairness & Bias Evaluation
* 📊 Model Transparency
* 🚨 Blind Spot Discovery
* 🛡️ Trustworthy AI for Security Applications

---

## 🎯 Objectives

* Detect phishing websites using advanced machine learning models.
* Analyze URL, domain, and webpage-based security indicators.
* Explain model predictions using SHAP and LIME.
* Evaluate fairness and potential bias in phishing detection.
* Discover model blind spots and error patterns.
* Build an interpretable cybersecurity analytics framework.
* Develop a deployment-ready phishing detection solution.

## 📊 Dataset

This project utilizes the **PhiUSIIL Phishing URL Dataset**, a large-scale dataset containing phishing and legitimate website samples with URL, domain, content, and behavioral features.

### Dataset Source
